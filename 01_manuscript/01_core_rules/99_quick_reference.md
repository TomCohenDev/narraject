# Quick Reference

Print this page. Keep it at the table.

## Building the Pool

| Source | Dice |
|--------|------|
| Stat Spend | 1 point = 1d6 |
| Basic Trait | +1d6 |
| Advanced Trait | +2d6 |
| Standard Gear | +1d6 |
| Quality Gear | +2d6 |
| Situational Bonus | +1d6 to +2d6 |
| **Step Die** | d8 / d10 / d12 |

---

## Counting Hits

**Hit** = Any die showing **5 or 6**

Compare total Hits to Difficulty.

---

## Difficulty Scale

| Hits | Difficulty |
|------|------------|
| 1 | Easy |
| 2 | Moderate |
| 3 | Challenging |
| 4 | Hard |
| 5 | Heroic |
| 6+ | Legendary |

---

## Reading the Step Die

| Result | Effect |
|--------|--------|
| **Highest in pool** | You gain 1 IP |
| **Lowest in pool** | GM gains 1 IP |
| **Shows 1 or 2** | GM gets free complication |
| **Tied** | No effect |

---

## IP Economy

### Earning IPs
- Accept GM Intrusion: +2 (give 1 away)
- Step Die highest: +1
- Exceptional moment: +1 (GM discretion)

### Spending IPs
| Cost | Effect |
|------|--------|
| 1 IP | Reroll Step Die |
| 2 IP | Add +1d6 after roll |
| 3 IP | Partial success on failure |
| 1-5 IP | Narrative Injection |

---

## Injection Costs

| Cost | Scope |
|------|-------|
| 1 IP | Flavor detail |
| 2 IP | Minor fact |
| 3 IP | Relationship / knowledge |
| 4 IP | Tactical advantage |
| 5 IP | Story connection |

---

## GM Intrusions

1. GM describes complication
2. GM offers 2 IPs
3. Player accepts (keep 1, give 1 away) OR refuses (spend 1 IP)

**Free intrusions** (no IPs): Step Die 1-2, Zero Hits
